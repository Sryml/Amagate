阿门是一个 Blender 扩展，提供各种实用工具用于 Blade 的 3D 开发。  
已实现和计划实现的功能包含：地图创建和导出，模型导出，动画导出等。。。

我希望这个工具能够帮助那些有兴趣的人们踏入 3D 世界的大门，这就是阿门。

## 📖 Usage

从菜单栏中选择`编辑 -> 偏好设置 -> 获取扩展`，然后点击右上角`从磁盘安装`并选择下载好的扩展 zip 文件。
